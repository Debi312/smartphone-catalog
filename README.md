
# Smartphone Catalog

Zara Challenge

## Tech Stack

- React
- TypeScript
- Webpack
- React Router
- Context API
- Jest
- ESLint
- Prettier

## Getting Started

Install dependencies:

```bash
npm install
```

Create a `.env` file based on `.env.example`.

Start development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```
